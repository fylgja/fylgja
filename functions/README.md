## Installation

You can install Fylgja Utilities via npm or other Node-based package managers like pnpm or bun:

```bash
npm install @fylgja/functions
```

Alternatively, you can use it directly via a CDN:

```html
<link href="https://cdn.jsdelivr.net/npm/@fylgja/functions/index.min.css" rel="stylesheet">
```

## Usage

Once installed, you can import the full package with:

```css
@import "@fylgja/functions";
```

Alternatively, if you only need specific parts of the functions.

You can import any function mentioned below, using:

```css
@import "@fylgja/functions/{FUNCTION_NAME}";
```
